# JAVA game engine
For more info check the wiki or check out the screenshots attached bellow. <br> 
In case you want to try and run the game yourself: <br>
## Run Without an IDE

### Requirements
- Installed **JDK 21** (check with `java -version`)
- No Maven installation needed (the project uses `mvnw.cmd`)

### Run the game
From the project root directory, run:

## Windows (Powershell)
```powershell
.\mvnw.cmd clean javafx:run
```
## Linux / macOS (bash/zsh)
```bash
chmod +x mvnw
./mvnw clean javafx:run
```
